The range() function

The range() function is often used in a for loop.
It specifies between which number of times the for instructions should repeat. 

                        
for i in range(3):
    print(i)
                        
                    
                        
    0
    1
    2

The range() function can also be given arguments specifying
the number to start and end the range count. 

                        
for i in range(5,8):
    print(i)
                        
                    
                        
    5
    6
    7
