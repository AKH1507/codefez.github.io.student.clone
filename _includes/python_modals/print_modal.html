Print

print() is a Python function for displaying text on the screen. Anything between the parenthesis is printed on the screen.
However! Python can only handle one datatype at a time. (see casting or f-strings for more info)
You can print any datatype. 

                            
                                print("Hello world!")
                                print(2+2) # calculates a result!
                                
                            
                        
                            
                                Hello World!
                                4
