Casting

Python can only handle one type of data at a time. Casting must be used in order to combine a second datatype. 

str()
converts data into a string datatype 
int()
converts data into a integer datatype 
float()
converts data into a real/float datatype 

                            
data_a = "Jessica"
data_b = 35
print(data_a + " is " + str(data_b) + " years old")
                            
                        
                            
Jessica is 35 years old
