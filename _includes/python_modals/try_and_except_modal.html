try and except

The try and except function test out the code you put under the try and will display a message that you choose under except. After using except, you can also add a specific error that will print out the message you wish for it to print out. These errors are known as python exceptions. Below I've used the ZeroDivisionError which occurs when you try to divide a number by 0. 




 try:
    numerator = int(input("Enter the numerator: "))
    denominator = int(input("Enter the denominator: "))
    result = numerator / denominator
    print(f"Result: {result}") #f strings are used here, they are in the cheatsheet
except ZeroDivisionError:
    print("Error: Cannot divide by zero!")
except ValueError:
    print("Error: Invalid input, please enter numbers only.")
