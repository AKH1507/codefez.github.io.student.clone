lists

Lists are used to store multiple items in a single variable. Using square brackets[ ] while writing a variable we can simply create a list. Lists can be done in all: Integers, Boolean, String and Float. 




  mixed_list = [True, False, 'Hello', 'World', 1, 34, 2.4, 4.5]
  # Contains Boolean, String, Int, Float
  
  print(mixed_list)
