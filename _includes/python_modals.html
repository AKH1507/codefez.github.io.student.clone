For Loops

a for loop allows us to iterate a list of instructions for a specified number of times. 

            
    for i in range(3):
        print("hi")

The i represents a local variable.
Imagine that the i stands for iteration.

The i can be replaced by another name such as x or word or name.

The only rule here is to not name the local variable with a variable already used in your code or one that is a reservered word such as print or for. 

            
    for number in range(3):
        print(number)

The range() function

The range() function is often used in a for loop.
It specifies between which number of times the for instructions should repeat. 

                        
        for i in range(3):
            print(i)
                        
                    
                        
        0
        1
        2

The range also means that the local variable (eg. i) will be represented as a number.
